FONT

  1. Font Family

    2. 글꼴을 지정할 때 font-family를 사용
    해당 문서는 font-family: arial, verdana, "Helvetica Neue", monospace;로 설정됨
    이 경우 앞에서 부터 순서대로 먼저 존재하는 글꼴을 사용한다.
    모든 글꼴이 없을 수 있으므로, 마지막에는 특정 글꼴이 아닌 글꼴분류이름을 사용한다

    • 글꼴 분류
    • serif : 장식이 존재하는 글꼴
    • sans-serif : 장식이 없는 글꼴
    • monospace : 고정폭 글꼴
    • fantasy : 화려한 글꼴
    • cursive : 필기계열의 글꼴
