FONT SIZE

사용 단위

  1. px
  2. em
  3. rem

px는 절대적인 크기, em, rem은 상대적인 크기를 나타냄
다른 단위를 쓸 이유가 없다면 rem을 쓰자!
fonr 크기를 사용자가 조정할 수 있기 때문!

-> 웹 접근성을 고려

실습

PX (16px)
REM (1rem)

사용자가 브라우저 설정에서 폰트 크기를 조정하게 되면

  • px는 고정
  • rem은 변함

html 태그의 기본 font-size가 16px이기 때문에 1rem을 사용하면 16px로 보여짐.
따라서 브라우저의 폰트 사이즈를 바꾸게 되면 html 태그의 font-size가 바뀌기 때문에 1rem으로 설정한 텍스트의 크기도 같이 바뀌게 됨
