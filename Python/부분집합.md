# 부분집합 구하기



```Python
arr = [1, 3, 5, 7, 9, 11]

#arr list에서 부분집합을 구한다

n = len(arr) # n : 원소의 개수

for i in range(1 << n): # 1 << n (== 2**n) 즉, 부분집합의 개수만큼 반복
    for j in range(n): # 원소의 수 만큼 비트 비교
        if i & (1 << j):
            print(arr[j], end=',')
        print()
```
